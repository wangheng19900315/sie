Add/Drop/Switch Courses

Students may add, drop or switch courses before prescribed deadline (maximum 2 courses for 4-week session and 3 courses for 5-week session). Students who want to add or switch courses need to make sure the courses applied for is still available.

Add Course: Add the course in online application system after payment before the summer session starts（maximum 2 courses for 4-week session and 3 courses for 5-week session）. After summer session starts, please download the Add Form, fill out the form then return it back to the Administration office.

Withdraw Course: Download the Withdraw Form, fill out the form and return it back to  the Administration Office and confirm about the refund

Switch Course: Switch the course on the Online Application System. Please note we do not accept switching course application after deadline.

Drop Course Deadline

Students can drop course before prescribed deadline. Please download the Drop Form (BJTU-4; ECNU-4; ECNU-5), fill out the form with the instructor’s signature as agreement of dropping course and return it back to the Administration Office.  The  course dropped won’t show on the transcript.

Please note that dropping course won’t involve any refund, for more information please read the Refund Policy.
