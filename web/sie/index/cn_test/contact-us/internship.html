岗位：项目助理(Program Assistant)

如果你善于沟通，考虑事情细致周道，外加一口流利的英语，那么这项工作绝对适合你。


实习要求：
1. 协助校区主管，参与到暑校日常的行政管理中，不夸张地讲，这个暑校被你们承包了。
2. 参与筹划文化体验活动和商业领袖项目，并且负责各项活动的组织和管理工作，无论你是说话脑洞大开还是做事井井有条，我们都看好你。
3. 与教授保持密切的交流和接触，并帮助教授解决所遇到的问题和困难，如果人人都献出一份爱，那么教授会跟你一样觉得中国就是美好的人间。
4. 对在校学员提供咨询和帮助，并向校区主管汇报相关问题，如果你能提出一些小建议，那再好不过了。
5. 完成校区主管下发的临时任务，我们只需要你保持一点儿执行力。
6. 如果你是北京上海的本地人，或者对北交大或者华东师范附近的衣食住行情况了如指掌，I WANT YOU，我们需要你这个包打听。
7. 我们还需要一个PA来兼职Resident Assistant（RA），负责学生住宿相关的管理工作。

 
你的收获：
1. 虽然你并不一定太在意，但是我们还是会为你提供一份不错的薪金作为对你付出的肯定。
2. 长时间与教授亲密友好的相处，相信他们也不会吝啬为你提供一份升学或是求职的推荐信。
3. Resident Assistant（RA）会获得我们提供的项目期间的免费住宿。不用谢，你值得拥有。
4. 和一个理想与经验并存的管理团队为伍，除了项目的运作经验你还能从方方面面学到更多。
5. 领导能力和沟通能力获得提升的同时，也锻炼了你的执行力和活动策划的才能。助你今后更快的融入职场。
6. 如果你喜欢聚会，喜欢摄影，喜欢跟人聊天，或者天生就是一个电脑极客，那么。。。为什么要压抑自己的天性啊，来吧！

 

岗位：教学助理（Teaching Assistant）

简而言之就是我们要找的是学霸，有良好的学科背景或助教经验的最好啦。
 
实习要求：
1.  我们希望你是相关专业本科大二以上年纪或者在读研究生。
2.  英语和学术知识对你来说不是问题的话，就可以和教授愉快得聊天了。
3.  每位TA负责协助一位教授的所有课程授课。
4.  TA主要负责协助教授教学工作的相关事宜，当然包括课堂考勤检查，课堂材料的准备，以及协助教授收发和批改作业，管理考试等。
5.  每门课程每天需提供一小时的office hour，为学员提供答疑。
6.  关注教授的教学进程和学生的学习状态，及时向校园主管提供反馈和建议。
7.  Writing Instructor需要由English Composition的TA来兼职，来帮助同学们解决写作上的问题，分享写作经验，并且接受学生的预约提供一对一辅导。
 
你的收获：
1. 我们会提供一份不错的薪水以表示对你付出的肯定和感谢。
2. 与教授长期的共事，或许会让教授为你今后的升学和求职提供一些帮助，比如一封得力的推荐信。
3. 表现优异者，我们除了提供奖金还可能提供一些实习推荐或者工作介绍。
4. 我们为Writing Instructor提供了免费的校内住宿。
5. 在与小伙伴的朝夕相处中，本身也会体会到一个帮助别人成就自己的快乐的过程。

 

岗位：校园大使 （Campus Representative）

热情开朗的性格，在周围同学中有超高的人气，或者特别会聊天儿，总之我们会为你提供一个展示的舞台。
 
实习要求：
1. 与学员建立联系，宣传暑校优势。
2. 配合线上的宣传活动，拓展市场渠道。
3. 按照部署，执行市场部制定的工作计划。
4. 组织宣讲会，推广宣传项目课程。
5. 搜集转学分相关信息，并及时反馈且提出建议。
6. 开拓和维护与本校学生组织之间的合作关系，巩固合作。
 
你的收获：
1. 销售方面的实践经验，让你逐步学着去串连所有的工作学习进程，把握处理事情的节奏，解决问题的方法。
2. 我们为你提供基本工资和绩效奖金，还可以获得SIE的实习证明及推荐信。
3. 在一个和同龄人的亲密交流的平台中，不难发现原来身边有这么多志趣相投的朋友。
4. 在工作中不断完善自己，执行能力，沟通能力，管理能力，表达能力的全面提升，获得成长的助力。
 

请发送你的简历至邮箱：hr@sieschool.org，注明姓名，所申请的校区（华东师范大学/北京交通大学）和申请职位。申请教学助理的同学请随简历附上相关科目的在校成绩单。
