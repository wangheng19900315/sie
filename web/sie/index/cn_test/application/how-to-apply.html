如何申请

北京交通大学/华东师范大学学生

北京交通大学/华东师范大学学生在本校教务处报名，学分作为选修课学分，可被计入学生在读学位的学分。教务处每年会在5月左右发出报名通知，请同学们密切关注教务处网站。

其他院校学生学生

申请流程图：

第一步：在线提交申请

在报名系统中完成四周课程或五周课程的线上申请表格，并在截止日期前提交申请。北京交通大学暑校四周课程正常申请截止日期是2016年6月19日。华东师范大学暑校四周课程正常申请截止时间为北京时间2016年5月15日；华东师范大学暑校五周课程正常申请截止时间为北京时间2016年6月19日。请参考（学费详情）获取更多有关学费和缴费截止日期的信息。

第二步：支付学费

需要缴纳的费用会根据您所选择的课程数量而有所变化，请参考学费详情获取更多有关学费和缴费截止日期的信息。

第三步：在线选课

学生在完成所有费用的支付，并在财务审核通过后，可以在申请系统中完成在线选课。如果所选课程已报满，请经常查看选课系统，以便在课程有空位时及时填报。

第四步：（外籍学生）申请中国签证

外籍学生需要纸质确认信方可申请入华签证。索要确认信，请发邮件至： iss@ied.ecnu.edu.cn （华东师范大学暑校）; 或 iss@sieschool.org （北京交通大学暑校）。对于短期海外求学项目，不必申请学生签证（X），可直接申请旅行签证（L）。
注意，中国大使馆并不接受邮寄申请签证的方式，因此若需申请中国签证，应到最近的中国大使馆提交相关表格及资料。如果本人不能亲自前往大使馆申请签证，可通过签证申请机构，申请付费代办。
