课程更改

加课/销课/换课

学生可以在规定截止日期之前进行加课、销课和换课（总课程数不能超过两门）。加课和换课的同学请确保新申请的课程仍有名额空缺。

加课：开学之前，请付费后在申请系统中添加课程（最多不超过两门），开学之后如需加课，请到暑校办公室填写《加课申请表》；

销课：申请销课的同学请到暑校办公室填写《销课申请表》，并咨询相关退费事宜。

换课：申请换课的同学请在网上报名系统上自行换课，请注意最后换课截止时间，过后不再接受换课申请。

退课截止日期

学生有权在规定截止日期之前进行退课。申请退课的同学请到暑校办公室填写《退课申请表》，并需任课教授签名同意；所退课程的信息不会出现在学生的成绩单上。

请注意：退课不涉及任何退费事宜。如需详情，请查询退款协议。
