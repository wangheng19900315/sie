申请流程


第一步：在线申请

请在报名系统中完成注册，并准确填写个人信息，选择所需的课程项目。

第二步：支付学费

支付所选的课程项目相应的学费，如选择校内住宿，住宿费需一并交纳。具体价格和优惠截止日期请参考学费价格，如有其他问题请直接与我们联系。

第三步：在线选课

支付费用后，审核会在1个工作日内完成，审核通过后即可在申请系统中继续完成选课。

注：

- 每门课程人数有限，请尽早完成“选课”以确保名额。

- 如果所选课程已报满，可以经常查看选课系统或直接联系我们，以便在课程有空位时能及时填报。
 

* 外籍学生需在上述步骤之后申请中国签证

外籍学生需要纸质确认信方可申请入华签证。索要确认信，请发邮件至： iss@ied.ecnu.edu.cn （华东师范大学暑校）; 或 bjtuiss@bjtu.edu.cn（北京交通大学暑校）。对于短期海外求学项目，不必申请学生签证（X），可直接申请旅行签证（L）。

注意，中国大使馆并不接受邮寄申请签证的方式，因此若需申请中国签证，应到最近的中国大使馆提交相关表格及资料。如果本人不能亲自前往大使馆申请签证，可通过签证申请机构，申请付费代办。
通过在网上搜索“Chinese Visa Service”可查询到此类机构。以下机构可作参考：

https://www.oasischinavisa.com/business_visa.html

http://www.mychinavisa.com/business.php

http://www.flychina.com/chinavisa.asp

也可查询中国大使馆官方网站获取更多签证申请信息 http://www.china-embassy.org/eng/

* 华东师范大学在校生可以在华师大教务处报名，所修课程将以选修课学分计入个人成绩，具体报名时间请密切关注本校教务处网站。
