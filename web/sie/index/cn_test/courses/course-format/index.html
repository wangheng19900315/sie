研究项目：科学家如何研究周边世界（非学分课程）


课程时间：2017年7月8日–7月29日，每周六 9:00–11:30
上课地点：北京交通大学（具体地点待定）
课程费用：非北京交通大学暑校在校生1500美元
                    北京交通大学暑校在校生1350美元
招生名额：18人（先到先得）
报名要求：高中及以上学历
课程教材：无指定教材
所需设备：个人电脑及相关软件（比如word、statistics）
项目简介：
本项目由美国顶尖科学家Edward Chang教授开设。在为期四周的项目中，学生将在这门有趣而又精彩的课程中了解并接触到全球众多社会科学家用来研究世界的技术与方法。学生将分成小组进行全新的科学实验项目，并在课程结束时进行汇报。除了能亲身体验和感受研究过程以外，学生还将学习如何申请国际知名的科学机构（比如Sigma Xi），为将来申请工作或研究生学习增添更辉煌的履历。
 
本项目可以为你供一次难得的机会师从知名的研究者进行科研学习，同时，让你从众多学生中脱颖而出。由于该课程是非学分课程，学生无需担心考试是否及格，从而更好地投入学习过程。本课程能够帮助您申请知名的科学机构，在国际科学大会上展示研究项目，甚至探索新的研究领域！

2016年暑校期间参加该研究项目的学生，不仅学到了严谨的科学研究方法，同时收获了许多乐趣。在项目结束之后，其中一些学生申请成为了Sigma Xi的成员，数项研究计划通过美国科学研讨会议的批准，还有一篇研究论文提交到一份国际知名研究杂志。同时，该项目为他们的履历增添了靓丽的一笔。
 
想要了解更多有关该项目的信息，请邮件联系Edward Chang教授：changec@umich.edu.
